# Variáveis

* Nomes simbólicos para receber algum valor
* Atalho de código
* Identificadores
* 3 palavras reservadas para cada variável
    * var -> varia
    * let -> varia
    * const -> não varia "constante"
    
```html
<script>

var clima = "Quente"
clima = "Frio"
console.log(clima)

</script>
```
